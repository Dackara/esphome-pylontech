# Changelog

## v1.0.0

First official public release.

### Added

* Native ESPHome external component
* Master mode support
* Slave mode support
* PWRSYS support
* PWR support
* GETPWR support
* INFO support
* STAT support
* BAT support
* SOH support
* Cell voltage monitoring
* Cell temperature monitoring
* Cell status monitoring
* Home Assistant integration
* Publish-on-change support
* Login Debug recovery support
* US2000B initialization support
* Optional PSRAM memory optimization
* UART queue protection
* UART queue diagnostics
* Protocol diagnostics
* Role validation diagnostics

### Notes

Current validation status:

* US2000C Master validated
* Slave validation in progress
* Other models not yet validated
