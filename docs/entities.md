# Entities

## Fast Loop Entities

Updated continuously.

### PWRSYS

Examples:

* Battery voltage
* Battery current
* Power
* Remaining capacity
* SOC

### PWR

Examples:

* Per-pack voltage
* Per-pack current
* Per-pack temperature

### GETPWR

Examples:

* Individual cell voltages
* Individual cell temperatures

---

## Slow Loop Entities

### INFO

Examples:

* Manufacturer
* Device name
* Firmware versions
* Board information
* Barcode
* Release date

### STAT

Examples:

* Alarm states
* Warning states
* Protection states

### BAT

Examples:

* Battery status
* MOS status
* Internal state information

### SOH

Examples:

* State of Health
* Cell status information

---

## Diagnostic Entities

Optional.

Examples:

* Protocol Online
* Response Time
* Consecutive Failures
* Timeout Count
* Last Command
* Last Error
* Configured Role
* Observed Role
* Role Validation

These entities are primarily intended for:

* debugging
* protocol validation
* troubleshooting

Most users do not need them.
